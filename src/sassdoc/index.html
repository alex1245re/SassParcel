    navigation

    @mixin navigation() { ... }

    Description

    Genera los estilos estructurales y visuales para un componente de navegación. Utiliza funciones para obtener colores y tamaños definidos en el módulo de variables global. Incluye estilos para la lista de navegación, los elementos y los enlaces, con transiciones para interactividad.

    Parameters

    None.

    Example

    Uso esperado del mixin

    .navbar {
    @include navigation();
}

    Requires

    • [function] color
    • [function] size
    • [function] color
    • [mixin] flexposition — Mixin de configuración de Flexbox.
    • [external] vars.color — Función que devuelve los colores del tema (primary, quaternary).
    • [external] vars.size — Función que calcula los tamaños de fuente (xs).

    Author

    • alejandro felipe

    media

    @mixin media($clave, $type) { ... }

    Description

    Mixin para media queries basado en los tamaños definidos en $sizes-media

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $clave

    Clave del tamaño (small, medium, large, xlarge, xxlarge) definida en $sizes-media

    		String— none
    $type

    Tipo de media query ('min' o 'max'), por defecto 'min'

    		String— none

    Example

    uso del mixin media

    .container {
  // Para min-width (mobile-first)
  @include mixins.media(medium, 'min') {
   max-width: 800px;
  }
  // Para max-width (aplicar estilos en pantallas pequeñas)
  @include mixins.media(medium, 'max') {
   padding: 1rem;
  }
}

    Requires

    • [variable] sizes-media

    Author

    • alejandro felipe

    cabeceras

    @mixin cabeceras() { ... }

    Description

    Genera los estilos tipográficos base para los encabezados del nivel 1 al 3 (h1, h2, h3). Extrae los valores de fuentes, tamaños y pesos del módulo de variables global.

    Parameters

    None.

    Example

    Uso del mixin cabeceras

    @include mixins.cabeceras;

    Requires

    • [function] size
    • [function] weight
    • [variable] font-heading
    • [external] vars.$font-heading — Familia tipográfica para encabezados
    • [external] vars.size — Función que calcula el tamaño base (sm)
    • [external] vars.weight — Mapa o función que devuelve el peso de la fuente (bold)
